# ValoStatus
**ValoStatus** is a **software** that **allows a user to get account and MMR data for a Valorant player**

# How to install
- **Desktop:**
    - **Linux | Mac OS:**
        - **Install** Python (Required Version: **3.14**)
            - For [Linux](https://docs.python-guide.org/starting/install3/linux/)
            - For [Mac OS](https://docs.python-guide.org/starting/install3/osx/)
        - **Download** ValoStatus CLI/GUI latest version's file
        - **Unzip** ValoStatus CLI/GUI latest version's file
        - **Install** ValoStatus CLI/GUI dependencies
            - ```pip3 install -r requirements.txt --user```
        - **Run** ValoStatus
            - ```python3 ValoStatus CLI/GUI.pyc```
    - **Windows:**
        - **Download** ValoStatus CLI/GUI latest version's file
        - **Unzip** ValoStatus CLI/GUI latest version's file
        - **Run** ValoStatus CLI/GUI.exe file

# How to use
- **Desktop:**
    - **CLI Version:**
        - Paste the Valorant player's username
        - Paste the Valorant player's tag
        - Choose the region (Use the up or down arrows to select the region)
    - **GUI Version:**
        - Paste the Valorant player's username into the first text field
        - Paste the Valorant player's tag into the second text field
        - Choose the region using the box next to the second text field
        - Click the "Search" button
- **Mobile:** (Not available)
